O que é um repositório no Git e qual a sua importância?
O repositório do Git é um repositório que serve para armazena arquivos, código e histórico de revisão de um projeto.

Qual a função do Git Bash?
O Git Bash é um aplicativo serve para criar e gerir versões de projetos no sistema operativo Windows. 

O que faz o comando git add e qual a diferença entre git add <arquivo> e git add .?
O git add serve para que você atualize as entradas da branch, o git add <arquivo> serve para atualizar arquivos especificos, já o git add . serve para atualizar todo os arquivos vinculados a branch.

Por que utilizamos o ponto (.) no comando git add .?
O ponto serve para que todos os arquivos do projeto sejam chamados de uma vez, facilitando o processo de git add.

Explique o comando git commit e por que é importante criar uma mensagem descritiva.
O git commit serve para que seja salvo um ponto na linha do tempo de um projeto, e as mensagens descritivas servem para explicar o que foi realizado neste ponto do tempo.

Qual a função do comando git push?
Serve para enviar alterações confirmadas (commits) de um repositório local para um repositório remoto. 

Qual a importância de criar commits frequentes durante o desenvolvimento de um projeto?
É importante para ter sempre um histórico das mudanças feitas no projeto, caso ocorra um erro em algum comito avançado, sempre é bom ter commits mais antigos para chegar a raiz do problema.

Explique a função das tags HTML e como elas estruturam uma página.
As tags HTML são elementos que definem a estrutura e o conteúdo de uma página web. Elas são usadas para criar títulos, parágrafos, imagens, links, entre outros que ajudaram na estruturação de uma pagina.

O que são atributos em uma tag HTML e qual a sua função?
Atributos em HTML são informações adicionais que se inserem na tag de abertura de um elemento. Eles controlam o comportamento, a aparência e a funcionalidade do elemento., como o class, que ajuda a classificar uma tag.

O que é o Emmet e como ele facilita o desenvolvimento de páginas HTML?
O Emmet é um plugin (para diversos editores/IDEs) que facilita bastante a escrita do HTML, provendo abreviações poderosas, snippets dinâmicos e diversos atalhos para escrever HTML.

O que é o box-sizing no CSS e por que usamos ele no reset inicial?
O box-sizing é utilizado para alterar a propriedade padrão da box model, usada para calcular larguras (widths) e alturas (heights) dos elementos.

Por que é importante utilizar tags semânticas no HTML?
As tags semânticas do HTML são importantes porque melhoram a acessibilidade, a otimização para mecanismos de busca e a legibilidade do código. 

O que são media queries no CSS e qual a sua função?
O media queries serve para que o site seja adaptavél para varios dispositivos,

Qual a função do GitHub Pages?
O Pages serve para que o seu site fique no ar de modo gratuito.

O que é o Flexbox no CSS e para que ele é utilizado?
O Flexbox é uma ferramenta do CSS que permite criar layouts flexíveis, distribuindo espaços e alinhando itens em um contêiner.

Qual a diferença entre justify-content e align-items em um container Flexbox?
O justify-content controla o alinhamento dos itens no eixo principal, enquanto o align-itens controla o alinhamento no eixo transversal. 

Como o flex-grow funciona e qual o seu propósito no layout Flexbox?
O flex-grow define a habilidade de um flex item de crescer, caso necessário, e ele dentro do Flexbox define a capacidade de um elemento crescer dentro de um layout.

Explique a diferença entre flex-direction: column; e flex-direction: row;.
O column serve para colocar os elementos em forma de colunas e o row serve para disponibilizar os elementos em forma de linhas.
